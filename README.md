# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on retail sales data to identify customer behavior, sales trends, and product performance. The analysis helps uncover meaningful insights that can support business decision-making and improve marketing strategies.

## Objective

The primary objective of this project is to:

* Analyze retail sales data.
* Understand customer demographics and purchasing behavior.
* Identify sales trends and patterns.
* Generate actionable business insights through data visualization and statistical analysis.

## Dataset Information

The dataset contains the following attributes:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
* Visual Studio Code (VS Code)

## Analysis Performed

1. Data Loading and Exploration
2. Data Cleaning and Validation
3. Descriptive Statistics
4. Gender Distribution Analysis
5. Age Distribution Analysis
6. Product Category Analysis
7. Sales by Gender Analysis
8. Monthly Sales Trend Analysis
9. Correlation Heatmap
10. Customer Spending Analysis

## Key Findings

* The dataset contains 1000 transactions with no missing values.
* Female customers contributed slightly more sales than male customers.
* Electronics generated the highest revenue among all product categories.
* The average transaction amount was ₹456.
* Monthly sales showed fluctuations, indicating varying customer demand throughout the year.
* Customer spending distribution was positively skewed due to high-value transactions.

## Business Recommendations

* Focus marketing efforts on high-performing product categories such as Electronics.
* Develop targeted campaigns for female customers.
* Introduce customer loyalty and retention programs.
* Analyze seasonal sales patterns for better inventory planning.
* Monitor high-value transactions to maximize profitability.

## Project Outcome

The project successfully identified customer purchasing patterns, sales trends, and category-wise performance. The insights generated can help businesses improve customer engagement, inventory management, and overall sales strategies.
